Dependencies

LuaxPL depends on the following modules;

  • Copas
  • LuaSocket
  • CopasTimer
  • Date

IMPORTANT: The current release (1.1.6) of Copas will not work because it does not support UDP. You should download the HEAD revision from the github repo, or use my version
