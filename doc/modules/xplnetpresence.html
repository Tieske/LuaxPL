Module xplnetpresence

Commandline utility to scan syslog output for MAC and IP addresses for domestic occupancy detection.

Domestic occupancy detection and notify through xPL. Use option -help for a description. &nbsp Example: xplnetpresence.lua -timeout=240 -port=53000 -instance=RANDOM -hub Example setup using a wireless router with DD-WRT firmware (version 'DD-WRT v24-sp2 (08/12/10) std');

  • Enable systemlog on tab 'service / services' of the admin console and set the target ip address where xplnetpresence.lua is listening.
  • On tab 'security / firewall' enable logging, set level 'low' and all three options (Dropped/Rejected/Accepted) to 'Enabled'.
  • Start xplnetpresence.lua using defaults
